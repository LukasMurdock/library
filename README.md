# Library

## Bookshelves

- 🏆 Favorites
- 📚 To Read
- Reading Now
- 📘 Have Read

## Scripts

### Open Local Library in Browser

```
./scripts/open.sh
```

### Search Google Books API to add book to library

```
./scripts/add.sh
```

### Build library

```
perl scripts/build.pl
```

### Deploy library

```
./scripts/deploy.sh
```
